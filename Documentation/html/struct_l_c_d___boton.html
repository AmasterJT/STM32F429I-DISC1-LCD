Detailed Description

Estructura para manejo de un botón.

Representa a un botón que se muestra en pantalla en una posición con una imagen de unas dimensiones, de forma que cuando se pulsa se ejecuta el código de una función. Se puede establecer un texto que se muestra sobre el botón.
