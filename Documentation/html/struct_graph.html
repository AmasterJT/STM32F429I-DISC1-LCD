Detailed Description

Estructura que representa los parámetros y propiedades de un gráfico.

Esta estructura contiene todos los parámetros necesarios para personalizar y dibujar un gráfico en pantalla. Permite configurar posición, tamaño, colores, y propiedades adicionales para el título, nombres de los ejes y cuadrícula.
