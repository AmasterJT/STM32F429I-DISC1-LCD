Detailed Description

Estructura de datos para un componente que maneja una imagen.

Permite la visualización de una imagen con unas dimensiones. Se puede establecer y modificar la posición, la habilitación de la imagen (si no está habilitada se muestra semitransparente y en blanco y negro), su visibilidad y se puede cambiar de imagen.
